BinHub

Take the frustration out of searching and focus on donating with BinHub


Mission

How can we better present the donation bin information, so that those in the greater Seattle area, especially college students, who would like to help the less fortunate can easily locate donation bins and find out more information about these donation bins in one central application?

Features

Map View

The map view will allow you to view donation bins around the user on one map

List View

The list view will show a list of donation bins around the user

Detailed View

The detailed view presents detailed information on each donation bin

Filter

Results can be better tailored to each user by filtering by donation type or donation organization

Verification

BinHub allows you to interact with the application by verifying donation bin locations and adding comments to each bin

Add a Bin

BinHub allows you to add new donation bins to the application as you find them
